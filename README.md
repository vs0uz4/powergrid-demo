# Livewire Powergrid Sample Application with Laravel

## Steps to Run
- git clone `git@github.com:vs0uz4/powergrid-demo.git`;
- cd powergrid-demo;
- composer install;
- npm install;
- npm run dev;
- php artisan migrate;
- php artisan db:seed;
- php artisan serve.

## How to Access
> Open URL (http://localhost) in your browser
